# 🌐 3D Portfolio Website

A modern, responsive **3D Portfolio Website** built using HTML, CSS, JavaScript and Three.js.  
Deployed using Firebase Hosting.

---

## 🚀 Live Demo
🔗https://my-portfolio-raushan.web.app/

---

## 📌 Features

- ✨ Modern dark UI with neon theme
- 🌐 Fully responsive (mobile + desktop)
- 🎯 Smooth scrolling navigation
- 🧑‍💻 About, Skills, Projects, Contact sections
- 📄 Resume download button
- 🖼️ Profile image integration
- 🎮 3D animated background using Three.js
- ⚡ Fast and optimized performance

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Three.js**
- **Firebase Hosting**
